# TopTier - Luxury Real Estate Platform

A premium real estate investment platform built with React, Node.js, and MongoDB.

## Features

- Modern dark-themed UI with glassmorphism design
- Property listings with advanced filtering
- Admin dashboard for property management
- User authentication system
- Responsive design for all devices

## Tech Stack

**Frontend:**
- React + Vite
- Tailwind CSS
- Framer Motion
- React Router

**Backend:**
- Node.js + Express
- MongoDB + Mongoose
- JWT Authentication
- Multer for file uploads

## Getting Started

### Prerequisites
- Node.js 18+
- MongoDB

### Installation

1. Install server dependencies:
```bash
cd server
npm install
```

2. Install client dependencies:
```bash
cd client
npm install
```

3. Start the development server:
```bash
cd server
npm run dev
```

4. Start the client:
```bash
cd client
npm run dev
```
